## 📌 Overview

A fully responsive, animated personal portfolio website showcasing my skills, projects, and experience as a **Data Engineer & ML Developer**. Built from scratch with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

> 🔗 **Live at:** [AbdellahKoulla.github.io](https://AbdellahKoulla.github.io)

---

## ✨ Features

- 🌗 **Dark / Light theme toggle** — saved to localStorage
- ⚡ **Smooth scroll animations** — fade-in on scroll with IntersectionObserver
- 🖱️ **Custom cursor** — interactive glow effect
- 📱 **Fully responsive** — mobile, tablet & desktop
- 🎨 **Animated hero** — morphing profile image + floating icons
- 🧭 **Active nav detection** — highlights current section while scrolling
- 🏷️ **Sections:** Hero · About · Skills · Projects · Experience · Contact
